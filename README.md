# whisper.

> *your private canvas for the noise inside*

---

Some thoughts are too loud for words.
Some moments are too quiet to lose.
Whisper is where they go.

Not a productivity app. Not a habit tracker. Not a streak counter.
Just a space — yours, offline, silent — where you can leave whatever needs leaving.

---

## what it is

A freeform mobile journal built with Flutter.

Each *whisper* is a blank canvas. You decide what goes on it —
a sentence, a scribble, a photo, a voice note, a song that was playing.
No templates. No prompts. No pressure.

Notebooks hold your whispers. That's the whole structure.
Simple on purpose.

---

## what you can put on a canvas

| | |
|---|---|
| ✍️ text | write in any color, any size |
| 🖊️ drawing | pen · pencil · marker · highlighter · eraser |
| 🖼️ photo | crop, scale, rotate freely |
| 🎙️ voice | record a voice note directly on the canvas |
| 🎵 music | attach a Spotify / Apple Music / YouTube link as a mood card |
| 🌙 background | 10 canvas colors — from washi to midnight |
| 📤 export | share as a high-quality image |
